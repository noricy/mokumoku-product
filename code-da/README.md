# code-da (コード打)

もくもく会用の寿司打風タイピングゲーム。コードやコマンドを打って「ドリンク代の元を取る」のがゴール。

## ローカル開発

```bash
npm install
npm run dev
```

http://localhost:5173 を開く。

## 本番ビルド

```bash
npm run build
npm run preview   # ローカルで配信して確認
```

`dist/` に静的ファイルが出力されるので、そのままどこにでも上げられる。

## 共有 (デプロイ)

### Vercel (おすすめ・1分)

```bash
npx vercel
```

対話に従うだけ。本番URLが返ってくるので Slack 等に貼って共有。

### GitHub Pages

リポジトリにこの `code-da/` を push して、`vite.config.ts` に `base: '/<repo-name>/'` を追加 → `npm run build` → `dist/` を `gh-pages` ブランチに公開。

### Netlify / Cloudflare Pages

ドラッグ&ドロップで `dist/` フォルダを投げ込めば公開できる。

## ゲームルール

エンジニア向け寿司打。詳細仕様は [SPEC.md](./SPEC.md)。

**対応プラットフォーム**: デスクトップ(物理キーボード)+ モバイル(ソフトキーボード + 画面上のアクションボタン)。

- 3 コース: お手軽 ¥300 / お勧め ¥500 / 高級 ¥1000
- 「お題」として表示されたコード/コマンドを正確にタイプして次のお題へ
- 1文字正タイプごとに ¥4〜¥6 即獲得 (コース別)
- IDE 風補完: タイプしたプレフィックスに前方一致するプール内候補が一覧表示
- `Tab` / `Enter` は **次のトークン1個だけ**補完 (target 全体を一気に埋めない)
- 引用符付きトークン (`"fix"` 等) や記号入りトークンは **literal** 扱いで補完不可。
  入力欄に「literal — Tab 不可」表示が出たら自力でタイプ
- `↑↓` で候補ナビ。target 以外を選んで `Tab` しても何も起きない (空打ち防止)
- `Shift+Tab` でリフレッシュ (バッファ空: お題ガチャ + ペナルティ / 非空: 入力クリア)
- `Backspace` で1文字戻る / `Esc` で中断
- コース代以上稼げば「元取れた」(ランク B 以上)

## お題の追加

`src/data/words.ts` の `CASUAL` / `RECOMMENDED` / `PREMIUM` に追記するだけ。
もくもく会で出てきたネタを足していくとウケる。
